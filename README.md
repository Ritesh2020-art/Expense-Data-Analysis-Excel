# Expense Data Analysis Dashboard (Excel Project)

## Project Overview

This project analyzes personal expense data using Microsoft Excel.
The objective of the project is to understand spending patterns and visualize expenses using charts and pivot tables.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Excel Charts
* Data Cleaning Functions

## Dataset Description

The dataset contains daily expense records with the following fields:

* Date
* Category
* Sub Category
* Payment Mode
* Amount
* Description
* Month

## Key Analysis

The project answers the following questions:

* What is the total expense each month?
* Which category has the highest spending?
* Which payment method is used the most?

## Dashboard Features

The Excel dashboard provides the following insights:

* Monthly Expense Trend
* Category-wise Expense Distribution
* Payment Mode Analysis
